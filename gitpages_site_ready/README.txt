index.html をブラウザで開くと、添付資料の内容を読めます。

GitHub Pages で公開する手順（最短）:
  1. GitHub で新規リポジトリを作成（Private でも可）
  2. このディレクトリ内のファイルをすべてアップロード
  3. リポジトリ Settings → Pages → Source を "Deploy from a branch"
     Branch: main / (root) を選び、Save
  4. 数分で https://ユーザー名.github.io/リポジトリ名/ が表示されます
